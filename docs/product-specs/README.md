# product-specs — 기능 명세

기능별 1문서. 새 기능 구현 전 여기에 명세를 쓰고, 변경 시 갱신한다.

## 명세 대상 기능 목록 (현재 구현됨 — 문서화는 점진적으로)

- 학습 흐름: 모드 선택(랜덤/순서/AI만/체크만) → 카드 학습 → Enter 단계별 노출
- 찾기: 입력 즉시 검색, 카드 점프(첫 화면 포함), 관리자 순서변경·단원이동
- 체크: 카드 체크 → 계정별 R2 동기화 → '체크한 카드만' 출제·인쇄
- 인쇄: A4 좌·우 2단, 목차, 전체/단원/체크범위 선택 (print.html + Paged.js)
- AI 생성: 토픽 → 정의·내용·키워드 자동 채움 (Gemini), 프롬프트 편집
- 관리자: 엑셀 업로드/다운로드, 카드 CRUD, 이미지(R2), 화이트리스트

작성 규칙: 사용자 시나리오 → 동작 정의 → 엣지 케이스 → 관련 코드 위치.
